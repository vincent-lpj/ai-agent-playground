# AI Agent Playground

A personal exploration and learning repository for experimenting with various AI agent frameworks. This repo contains sample implementations and integration examples for different agent development platforms.

## 📁 Repository Structure

```
ai-agent-playground/
├── google-adk/          # Google Agent Development Kit explorations
├── langchain/           # LangChain framework experiments
└── ms-autogen/          # Microsoft AutoGen framework exploration
```

## 🎯 Purpose

This repository serves as a personal sandbox for:

- **Learning** different AI agent frameworks and their patterns
- **Experimenting** with agent capabilities (memory, tools, workflows)
- **Comparing** framework approaches and trade-offs
- **Storing** reusable examples and integration patterns

## 🛠️ Frameworks Explored

### Google Agent Development Kit

- Foundational agent implementation patterns
- Agent capabilities and configuration

### LangChain

- Multi-agent orchestration
- Integration with local LLMs via Ollama
- Agent memory management
- Human-in-the-Loop (HITL) workflows
- Model Context Protocol (MCP) integration

### Microsoft AutoGen

- Multi-agent conversation patterns
- Local LLM integration with Ollama
- Agent collaboration and communication

## 🚀 Getting Started

### Prerequisites

- Python 3.8+
- Virtual environment setup (see `pyproject.toml`)
- Optional: [Ollama](https://ollama.com/) for running local LLMs

## 📝 Notes

- This is a **personal learning repository**, not a production-ready library
- Code examples are intended for exploration and understanding
- Different frameworks may use different dependency versions
- See individual `README.md` files in each framework directory for specific setup instructions
