Integration: [Ollama](https://docs.langchain.com/oss/python/integrations/chat/ollama)

```bash
# Installation
uv add langchain
uv add langgraph
uv add langchain-ollama
uv add ipython

# To use checkpoint and store (PostgreSQL)
uv add psycopg langgraph-checkpoint-postgres

# To use MCP Adapters
uv add langchain_mcp_adapters
```
